This registry is the authoritative record of planned, active, and completed APS engineering increments.

## Increment Lifecycle

| State | Meaning |
| --- | --- |
| Proposed | Identified but not dependency-ready |
| Ready | Dependencies satisfied |
| Active | Currently being implemented |
| Completed | Repository evidence exists |
| Blocked | Requires business, technical, or governance decision |
